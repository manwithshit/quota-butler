飞书里的 AI 额度助手

额度管家

在飞书里看额度、排预热、收提醒，把 Claude Code 和 Codex 的可用时间安排到真正工作的时刻。

Pain Point

核心痛点：怎样最大化利用已经付费的 AI 额度。

看不见

5 小时窗口、周额度、刷新时间散在不同地方，真正开工时才发现关键额度已经见底。

容易错过

额度常在凌晨、开工前或会议中恢复；人没有盯着，就很容易白白错过可用窗口。

不在电脑前

需要安排预热时，人可能在路上、在开会、只拿着手机；最顺手的入口应该就在飞书。

How It Helps

从飞书私聊，把额度变成可安排的工作时间。

开工前先看清楚

直接知道 Claude Code / Codex 还剩多少，以及下一次什么时候刷新。

需要时立刻续上

临时想用就立即预热一次，不必创建长期计划，也不用切到电脑前操作。

明天重度使用

只告诉它开工时间，系统帮你排好预热节点，让可用窗口尽量贴近工作节奏。

过程随时可追踪

查看已执行、未执行、失败和下一次预热；不需要了，就取消所有计划。
